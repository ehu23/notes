# Homebrew Basics

## Before Anything:

`brew update`

followed by

`brew doctor`

and if you want to upgrade all formulas:

`brew upgrade`

## Normal Brew Installs:

`brew install TOBEBREWED`

## GUI Application Installs:

`brew cask install NAMEOFAPP`

## How to Start/Stop Software Services:

`brew services start SERVICENAME`
`brew services stop SERVICENAME`
